# Network-Enumeration
This Project demonstrates network enumeration techniques using Linux tools. It includes scanning, service detection , and information gathering to identify hosts, open ports , and potential vulnerabilities in a network envoirement.

## Steps Performed
1. Set up Kali linux in a virtual machine envoirement.
2. Identified the target system IP address.
3. Used Nmap to perform network scanning
4. Scanned for open ports on the target system
5. Identified running services on open ports
6. Performed service enumeration for more details
7. Analyzed scan results to understand network structure
8. Documented finding from the enumeration process

## Tools Used
-Kali Linux
-Nmap
